## MAC
1. 레디스 실행
> redis-server
> redis-cli 로 레디스 확인

2. kafka 실행
> /opt/homebrew/opt/kafka >> brew Kafka 경로 \
> bin git:(stable) kafka-server-start ../libexec/config/kraft/server.properties

3. DB 시작
> brew services start mariaDB \ 
> sudo mysql -u root -p >> 터미널에서 디비 접속

4. 맥 터미널 경로
> / << homebrew, opt, application
> ~ << 바탕화면
